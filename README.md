# MedGuard AI

## Description

AI-powered Drug-Drug Interaction and Synergism Predictor

## Tech Stack

| Component | Technology (Proposed/Placeholder) |
| :--- | :--- |
| **Frontend** | React / Vue, TailwindCSS, Vite |
| **Backend** | FastAPI / Node.js |
| **DB** | PostgreSQL, MongoDB |
| **Mobile** | React Native / Flutter |
| **AI** | PyTorch, HuggingFace, Scikit-learn |

## Getting Started

```bash
# 1. Clone the repository
git clone https://github.com/krunalsakpal679-hue/medguard-ai.git
cd medguard-ai

# 2. Setup Backend
cd backend
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
# Start backend server
# uvicorn app.main:app --reload

# 3. Setup Frontend
cd ../frontend
npm install
# Start frontend server
# npm run dev

# 4. Setup Mobile
cd ../mobile
npm install
# Start mobile app
# npx expo start

# 5. Setup Admin
cd ../admin
npm install
# Start admin dashboard
# npm run dev
```
